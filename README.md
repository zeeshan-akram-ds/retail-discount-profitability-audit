# Discount Profitability & Revenue Leakage Audit

Retail businesses widely use discounts to drive sales, but often lack visibility into whether those discounts generate profit or silently erode margins.

This project analyzes transaction-level retail data to quantify:
- Where discounts are destroying profitability
- Which customers are dependent on discounts but not valuable
- The true cost of current discounting strategies in monetary terms

The goal is to move from revenue-focused reporting to profit-aware decision making.
